Tooltip for Division items

Items linked to kokarn.github.io/division-item-display/ will display a basic hover tooltip for easy viewing. It will only be displayed for single items, not pages including more than one item.

Rapid Assault Pads

Enhanced ACR-E

Collection of items, eg daily vendors or something like that

Usage

Included in a site

If you own the site you can add the script to your site to create the tooltips for all users. It's been tested on modern browsers and shouldn't break anything.

The script is availabe in the repository or if you like, you can use rawgit to link it directly.

Browser plugin

If you don't own the site (eg reddit), you can install the tooltip as a browser plugin
